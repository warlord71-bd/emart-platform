# Business outcome first

Owner preference recorded 2026-06-24:

- Every Emart task must start with the business outcome, not the artifact being produced.
- Content is not complete because it is well written. It must connect qualified search/social intent
  to useful product discovery, trust, conversion, repeat purchase, or another measurable outcome.
- SEO work must check query ownership/cannibalization, current GSC evidence, canonical destination,
  live relevant products, internal links, metadata, schema/markup, snippet readiness, and a clear
  conversion path before being called ready.
- Automation work must show sales impact, cost/time savings, risk reduction, or service improvement.
- Do not force sales language where it harms trust. Help the shopper decide, then provide a clear,
  accurate next action using current Emart data.
- Reports should state the intended KPI and how results will be measured after release.
